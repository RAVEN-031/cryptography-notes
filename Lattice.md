
Given a set of linearly independent vectors within a vector space, an integer lattice L spanned by the set of vectors **is the set of integers linear combinations of the given vectors.**
The set of vectors $B = {v_1, ..., v_n}$ is called the basis with $L = L(B)$ spanned by B. Dimension is defined by $dim(L) := n$
