# Knapsack

A knapsack is a set of different nonzero natural numbers. If such set can be arranged in an increasing list, where every number is larger than sum of previous number, it is called a **superincreasing knapsack**.

# Public Key

- Hard Knapsack, used only for encryption.

# Private Key

- Superincreasing Knapsack
- Multiplier $n$
- Modulus $m$
